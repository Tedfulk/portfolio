Tedle

A wordle clone

Tedle is a clone of the popular game wordle. Done all in HTML, CSS, and JS. This game calls a random word using a random word API. Then checks the users input to a dictionary API to make sure it is a real word. If so then the game logic is carried out.

The letter turns green if it is in the correct position, yellow if the letter is in the word but not in the correct position, and gray if the letter isn't in the word.
